Event-Organization-Dapp

Steps to run: 
1. Install metamask 
2. Select ethereum test network sepolia and add ethereum to it from sepolia faucet.
3. Open remix ide and compile the contract. 
4. select injected web and deploy the contract.
5. Get the deployed acccount id and put it in the account section in app.js file 
6. node and npm must be install , install express - npm install express.
7. Run the html file on the browser.
